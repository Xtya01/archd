# Archive File Manager

A **self‑hosted personal cloud file manager** using **Archive.org as storage**.

Think **Google Drive + Media Server**, but self‑hosted.

---

## Features

✅ Local file uploads  
✅ URL uploads  
✅ Upload queue with live status  
✅ Folder‑based file manager  
✅ Media streaming (MP4 / MKV / MP3 / M3U8)  
✅ Secure WebDAV (Basic Auth)  
✅ Mobile‑friendly UI  
✅ Docker + Portainer ready  
✅ Cloudflare streaming proxy
